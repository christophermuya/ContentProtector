Version 2.0.0

Content protector prevent users from performing certain action on a given node.

  • indicates that some nodes are protected.
  • indicates that there is a user exception.
  • indicates that all nodes will be protected, regardless of the selection.

Actions:

  • Save
  • Publish
  • Unpublish
  • Rename
  • Move
  • Copy
  • RollBack
  • Delete
  • Trash
